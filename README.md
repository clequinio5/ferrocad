# Ferrocad

Diagram editor project with project and component tree build with ReactJS, AntD and JointJS

# Install

npm install
npm start

# Dependancies

- Nodejs v13 (for node-sass compatibility)
- ReactJS
- Ant Design
- JointJs
- Recoil (for state management system)
